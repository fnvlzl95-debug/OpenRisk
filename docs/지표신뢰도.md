# OpenRisk v2 지표별 신뢰도 평가

> 마지막 업데이트: 2026-01-03

---

## 종합 신뢰도

| 항목 | 수치 |
|------|------|
| **데이터 기반 지표** | 80-85% |
| **추정 기반 지표** | 50-60% |
| **전체 평균** | **약 65-70%** |

## 지표별 상세 평가

| 지표 | 신뢰도 | 데이터 출처 | 비고 |
|------|--------|-------------|------|
| **경쟁 밀도** | 85-90% | 소상공인시장진흥공단 (2025.10) | 전수 조사, 공식 데이터 |
| **생존 신호** | 80-85% | 소상공인시장진흥공단 (2024.12 vs 2025.10) | 10개월 비교, 계절 변동 미반영 |
| **앵커 시설** | 90-95% | 지하철 DB + 카카오 POI API | 공식 데이터 + 실시간 API |
| **상권 성격** | 75-80% | 업종 분포 기반 추론 | 실제 소비 패턴 미반영 |
| **시간대 특성** | 60-70% | 지하철 시간대별 승하차 | 서울 실데이터, 경기/인천 추정 |
| **유동인구 지수** | 50-60% | 지하철+버스 승하차 추정 | 도보/자차 유동인구 미반영 |
| **임대료** | 40-50% | 구 평균 추정 | 블록별 편차 클 수 있음 |

## 높은 신뢰도 지표 (80% 이상)

### 1. 경쟁 밀도 (85-90%)

**강점:**
- 소상공인시장진흥공단 공식 전수 조사 데이터
- 2025.10 최신 데이터 사용
- 카카오맵(45개) 대비 3배 이상 포괄적 (이태원 양식점: 153개)

**한계:**
- 배달전문점, 공유주방 등 실제 경쟁자로 체감 안 될 수 있음
- 폐업 신고 지연으로 유령 점포 포함 가능

### 2. 생존 신호 (80-85%)

**강점:**
- 동일 출처 2개 시점 비교로 일관성 확보
- 개업/폐업 트렌드 객관적 산출

**한계:**
- 10개월 비교로 계절 변동 완전히 반영 못함
- 업종 변경/휴업 구분 불가
- 이전(위치 변경)을 폐업+개업으로 이중 카운트 가능

### 3. 앵커 시설 (90-95%)

**강점:**
- 지하철역: 공식 DB, 위치 정확
- 스타벅스/마트/백화점: 카카오 POI 실시간 검색

**한계:**
- 신규 개점/폐점 반영 시차 (1-2주)

---

## 중간 신뢰도 지표 (60-80%)

### 4. 상권 성격 (75-80%)

**강점:**
- 업종 구성 데이터 기반 객관적 분류
- A_주거/B_혼합/C_상업/D_특수 4단계 구분

**한계:**
- 업종 구성 ≠ 실제 소비 패턴
- 같은 "상업지역"도 성격이 다를 수 있음 (명동 vs 홍대)

### 5. 시간대 특성 (60-70%)

**강점:**
- 서울 247개 역: 실제 시간대별 승하차 데이터
- 출퇴근형/상업형/혼합형 패턴 구분

**한계:**
- 경기/인천: 일 승객수 기반 패턴 추정
- 지하철 이용객 ≠ 해당 상권 유동인구

---

## 낮은 신뢰도 지표 (60% 이하)

### 6. 유동인구 지수 (50-60%)

**강점:**
- 지하철+버스 승하차 데이터 활용
- 상대적 비교 지표로는 유효

**한계:**
- 자가용/도보 유동인구 미반영
- 환승객 이중 카운트 가능
- 실제 상권 내 체류 인구와 차이

**예시:**
- 강남역: 지하철 승하차 많지만 환승도 많음
- 가로수길: 지하철역 없지만 유동인구 많음

### 7. 임대료 (40-50%)

**강점:**
- 구 단위 평균 제공으로 대략적 수준 파악

**한계:**
- 같은 구 내에서도 편차 극심
  - 강남구: 강남역 1층 vs 개포동 주택가
  - 마포구: 홍대입구 vs 상암동
- 층수/면적/권리금 미반영
- 실거래 데이터 아닌 추정치

## 신뢰도 향상 로드맵

### 즉시 개선 가능

| 항목 | 방법 | 예상 효과 |
|------|------|-----------|
| 시간대 특성 | 경기/인천 버스 시간대 데이터 추가 | 60-70% → 70-75% |
| 상권 성격 | 매출 데이터 반영 | 75-80% → 80-85% |

### 외부 데이터 연동 필요

| 우선순위 | 데이터 | 비용 | 예상 효과 |
|----------|--------|------|-----------|

| 2 | 통신사 유동인구 | 유료 | 유동인구 50%→90% |
| 3 | 카드사 결제 데이터 | 유료 | 매출 추정 신뢰도 대폭 향상 |

## 사용자 가이드

### 신뢰할 수 있는 정보
- 주변에 같은 업종이 몇 개 있는지 (경쟁 밀도)
- 최근 점포가 늘고 있는지 줄고 있는지 (생존 신호)
- 지하철/대형마트가 가까운지 (앵커 시설)

### 참고 수준 정보
- 대략적인 상권 유형 (상권 성격)
- 시간대별 유동 패턴 (시간대 특성)

### 반드시 현장 확인 필요
- 실제 유동인구 (직접 방문 카운팅 권장)
- 임대료 (부동산 문의 필수)

## 리스크 점수 산출 체계

### 점수 해석

| 점수 | 등급 | 의미 |
|------|------|------|
| 70~100 | VERY_HIGH | 매우 위험 - 신중한 검토 필요 |
| 50~69 | HIGH | 위험 - 리스크 요인 다수 |
| 30~49 | MEDIUM | 주의 - 일부 리스크 존재 |
| 0~29 | LOW | 안전 - 양호한 조건 |

### 점수 산출 방식

```
종합 점수 = Σ(개별 지표 점수 × 가중치) + 시간대 보정 + 상권 유형 패널티
```

#### 1. 개별 지표 정규화 (0~100)

| 지표 | 산출 기준 | 상태 표현 |
|------|-----------|----------|
| 경쟁 | 동종 0~15개 → 여유~포화~과포화 | 과포화 = 위험 |
| 임대료 | 40만원/평 이상 → 고부담 | 고부담 = 위험 |
| 폐업률 | 15% 이상 → 불안정 | 불안정 = 위험 |
| 유동인구 | 100 이상 → 활발 | 저조 = 위험 |
| 앵커 | 10개 이상 → 집객력 충분 | 부족 = 위험 |

**경쟁 상태 기준:**
- 여유 (0~5개): 경쟁 적음
- 보통 (6~10개): 적정 경쟁
- 포화 (11~15개): 경쟁 과열
- 과포화 (16개+): 공급 과잉

**앵커 시설 카운트:**
- 지하철역: 1개
- 스타벅스: 실제 개수 반영
- 대형마트: 1개
- 백화점: 1개
- 10개 이상 = 집객력 충분 (안전)

#### 2. 업종별 가중치

업종 특성에 따라 각 지표의 중요도가 다름:

| 업종 | 경쟁 | 임대료 | 폐업률 | 유동인구 | 앵커 |
|------|------|--------|--------|----------|------|
| 카페 | 35% | 20% | 15% | 20% | 10% |
| 음식점 | 30% | 25% | 20% | 15% | 10% |
| 편의점 | 25% | 15% | 20% | 25% | 15% |

#### 3. 상권 유형 패널티

상업/특수 상권은 경쟁이 치열하고 임대료가 높으므로 추가 리스크 부과:

| 상권 유형 | 패널티 | 설명 |
|-----------|--------|------|
| A_주거 | 0점 | 주거 밀집, 경쟁 적음 |
| B_혼합 | +3점 | 직주 혼합 |
| C_상업 | +10점 | 상업 중심, 경쟁 치열 |
| D_특수 | +15점 | 역세권 핫플, 대학가 등 |

#### 4. 시간대 패턴 보정 (±10점)

업종별 최적 시간대와 실제 피크타임 매칭 여부:
- 일치 시: -5점 (리스크 감소)
- 불일치 시: +3~8점 (리스크 증가)

### 점수 예시

```
강남역 카페 분석:

1. 개별 점수:
   - 경쟁(158개): 100점 × 0.35 = 35
   - 임대료(높음): 100점 × 0.20 = 20
   - 폐업률(14%): 90점 × 0.15 = 13.5
   - 유동인구(높음): 20점 × 0.20 = 4
   - 앵커(지하철 50m): 10점 × 0.10 = 1

2. 기본 점수: 35 + 20 + 13.5 + 4 + 1 = 73.5

3. 보정:
   - 시간대 불일치: +3
   - D_특수 상권: +15

4. 최종: 73.5 + 3 + 15 = 91.5 → 92점 (VERY_HIGH)
   (단, 100점 상한으로 78점 출력)
```

### 유동인구 최소 점수 정책

유동인구가 많은 상권(역세권 등)이라도 경쟁/임대료 리스크가 존재하므로, 유동인구 지표의 최소 점수를 20점으로 설정:

- **이전**: 유동인구 80 이상 → 0점 (리스크 완전 상쇄)
- **현재**: 유동인구 80 이상 → 20점 (최소 리스크 유지)

이를 통해 "사람 많으니까 괜찮겠지"라는 과도한 낙관을 방지.
