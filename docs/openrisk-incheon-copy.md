# OpenRisk Incheon 문구 모음

이 문서는 현재 코드 기준으로 OpenRisk Incheon 화면과 API 응답에서 쓰는 한국어 문구를 한곳에 모은 것입니다.

대상 범위:
- `/incheon`
- `/incheon/result`
- `/incheon/map-demo`
- `components/incheon/*`
- `app/api/incheon/*`
- `lib/incheon/*`

표기 규칙:
- `{값}`은 API 계산 결과나 선택 업종에 따라 바뀌는 동적 값입니다.
- 현재 결과 화면에서 직접 노출하지 않는 내부 evidence/facts 문구도 별도 섹션에 포함했습니다.
- 원천 데이터 출처명과 설명 문구도 출처 패널에서 노출될 수 있어 포함했습니다.
