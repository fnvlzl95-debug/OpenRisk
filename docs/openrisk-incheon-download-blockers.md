# OpenRisk Incheon 다운로드 필요 목록

아래 자료는 자동 다운로드가 막히거나 인증키·조건 선택이 필요할 수 있습니다.

| 자료 | 제공기관 | 필요한 항목 | 필요한 조치 |
|------|----------|-------------|-------------|
| 어린이집 기본정보 API | 보건복지부 | 시설명, 위도, 경도, 정원, 현원, 입소대기, 연령별 아동 수 | 공공데이터포털 인증키 또는 파일 추출본 제공 |
| 행정동별 성별·연령별 주민등록 인구 | 행정안전부 | 행정동 코드, 행정동명, 0-9세, 10-19세, 65세 이상, 기준월 | 인천 행정동 단위 최신 월 CSV 또는 XLSX 제공 |
| 한국부동산원 소규모상가 공실률 | 한국부동산원 | 권역명, 공실률, 분기, 상가유형 | 인천 포함 소규모상가 공실률 CSV 또는 XLSX 제공 |
| 행정동·시군구 경계 데이터 | 행정안전부 또는 통계청 | 행정동 코드, 행정동명, 시군구명, geometry | 인천 행정동 경계 GeoJSON 또는 SHP 묶음 제공 |

현재 로컬 재사용 후보:

- `data/bus/incheon/ridership_2025.csv`
- `data/subway/incheon/stations.csv`
- `data/rent/commercial_rent_2025Q3.xlsx`
- `data/incheon_taxi_h3_data.json`
