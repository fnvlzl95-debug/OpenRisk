## 2. 경쟁 밀도 신뢰도

### 2.1 현황
| 항목 | 상태 |
|------|------|
| 데이터 소스 | 소상공인시장진흥공단 |
| 갱신 주기 | 분기 |
| 커버리지 | 서울 100%, 경기 100%, 인천 100% |

### 2.2 강점
- **공식 등록 데이터**: 사업자등록 기반 정확한 점포 수
- **업종 분류 체계**: 대분류/중분류/소분류 구조
- **H3 그리드 집계**: 일관된 공간 분석

### 2.3 한계점

| 문제 | 영향도 | 설명 |
|------|--------|------|
| 업종 분류 오차 | 중 | "카페"가 "음식점"으로 등록된 경우 |
| 폐업 반영 지연 | 중 | 실제 폐업 후 1~3개월 지연 |
| 무등록 점포 | 하 | 비공식 영업점 미포함 |

### 2.4 신뢰도 계산

```
신뢰도 = 기본 90% - 업종 오차 5% - 지연 5% = 80%
```

## 3. 유동인구 신뢰도

### 3.1 현황

| 지역 | 지하철 | 버스 | 종합 |
|------|--------|------|------|
| 서울 | ✅ 완전 | ✅ 완전 | ⭐⭐⭐⭐ |
| 인천 | △ 역 위치만 | ✅ 완전 | ⭐⭐⭐ |
| 경기 | ❌ 없음 | ✅ 완전 | ⭐⭐ |

### 3.2 데이터 구성

| 데이터 | 파일 | 기준 |
|--------|------|------|
| 서울 지하철 | `subway/seoul/ridership_2023.csv` | 2023년 |
| 서울 버스 | `bus/seoul/ridership_2022.csv` | 2022년 |
| 인천 버스 | `bus/incheon/ridership_2025.csv` | 2025년 |
| 경기 버스 | `bus/gyeonggi/ridership.csv` | 혼합 |

### 3.3 한계점

| 문제 | 영향도 | 설명 |
|------|--------|------|
| **경기 지하철 누락** | 상 | 경의중앙선, 분당선, 신분당선 등 누락 |
| 보행자 미반영 | 중 | 실제 보행 유동인구 추정 불가 |
| 승용차 미반영 | 중 | 차량 유입 고객 미포함 |
| 시간대별 추정치 | 하 | 서울 지하철만 시간대별 데이터 |

### 3.4 경기권 지하철 누락 상세

| 노선 | 주요 역 | 영향 지역 |
|------|--------|----------|
| 경의중앙선 | 일산, 파주, 고양 | 일산 일대 |
| 분당선 | 분당, 수원, 죽전 | 성남, 용인 |
| 신분당선 | 판교, 정자, 광교 | 성남, 수원 |
| 경춘선 | 춘천, 가평 | 가평, 남양주 |
| 수인선 | 인천~수원 | 시흥, 안산 |

### 3.5 신뢰도 계산

```
서울: 기본 95% - 보행자 미반영 10% = 85%
경기: 기본 95% - 지하철 누락 30% - 보행자 10% = 55%
인천: 기본 95% - 승하차 미반영 15% - 보행자 10% = 70%
```

## 4. 임대료 신뢰도

### 4.1 현황

| 항목 | 상태 |
|------|------|
| 데이터 소스 | 상가정보시스템 |
| 파일 | `commercial_rent_2025Q3.xlsx` |
| 단위 | 구/동 평균 (만원/평) |

### 4.2 한계점

| 문제 | 영향도 | 설명 |
|------|--------|------|
| **구/동 평균만 제공** | 상 | 같은 동 내 대로변 vs 골목 차이 큼 |
| 권리금 미포함 | 상 | 실제 창업 비용의 큰 부분 누락 |
| 건물 유형 미구분 | 중 | 1층 vs 지하, 신축 vs 노후 |
| 분기별 갱신 | 하 | 급등/급락 반영 지연 |

### 4.3 실제 오차 예시

| 지역 | DB 평균 | 실제 범위 | 오차 |
|------|---------|----------|------|
| 홍대 메인거리 | 40만원/평 | 50~80만원 | -40% |
| 홍대 뒷골목 | 40만원/평 | 15~25만원 | +60% |
| 강남역 | 45만원/평 | 30~100만원 | ±100% |

### 4.4 신뢰도 계산

```
신뢰도 = 기본 80% - 위치 오차 25% - 권리금 15% = 40%
```

## 5. 생존율 신뢰도

### 5.1 현황

| 항목 | 상태 |
|------|------|
| 데이터 소스 | 소상공인시장진흥공단 |
| 계산 방식 | (폐업 수 / 이전 기간 점포 수) × 100 |
| 커버리지 | 서울/경기/인천 100% |

### 5.2 강점
- 공식 폐업 데이터 기반
- H3 그리드별 집계로 지역 특성 반영
- 업종별 폐업률 세분화 가능

### 5.3 한계점

| 문제 | 영향도 | 설명 |
|------|--------|------|
| 폐업 원인 미구분 | 중 | 경쟁 폐업 vs 은퇴 vs 이전 구분 불가 |
| 휴업 미반영 | 중 | 장기 휴업 점포는 영업 중으로 카운트 |
| 프랜차이즈 체인 | 하 | 본사 정책 폐업은 지역 특성 아님 |

### 5.4 신뢰도 계산

```
신뢰도 = 기본 95% - 원인 미구분 10% - 휴업 5% = 80%
```

### 6.2 강점

- 거리 계산 신뢰성

### 6.3 한계점

| 문제 | 영향도 | 설명 |
|------|--------|------|
| 신규 개점 반영 | 하 | 신규 역/매장 수동 추가 필요 |
| 출구 정보 불완전 | 하 | 일부 역 출구 좌표 누락 |

### 6.4 신뢰도 계산

```
신뢰도 = 기본 100% - 갱신 지연 5% = 95%
```

## 7. 종합 리스크 점수 신뢰도

### 7.1 계산식

```
종합 신뢰도 = Σ(지표 신뢰도 × 지표 가중치)
```

### 7.2 업종별 종합 신뢰도 예시

| 업종 | 경쟁 | 유동 | 임대 | 생존 | 앵커 | 종합 |
|------|------|------|------|------|------|------|
| 카페 (서울) | 80%×35% | 85%×20% | 40%×20% | 80%×20% | 95%×5% | **71%** |
| 카페 (경기) | 80%×35% | 55%×20% | 40%×20% | 80%×20% | 95%×5% | **65%** |
| 한식 (서울) | 80%×30% | 85%×15% | 40%×30% | 80%×20% | 95%×5% | **65%** |
