## 상세 정보

### 1. 점포 데이터 (`grid_store_counts`)

**기준**: 2024년 10월 (202410)

**출처**: 소상공인시장진흥공단 상가(상권)정보

**원본 파일**:
- `소상공인시장진흥공단_상가(상권)정보_서울_202510.csv`
- `소상공인시장진흥공단_상가(상권)정보_경기_202510.csv`
- `소상공인시장진흥공단_상가(상권)정보_인천_202510.csv`

**포함 정보**:
- 업종별 점포 수 (21개 카테고리)
- 개/폐업 수 (전 분기 대비)
- H3 그리드 (해상도 9) 기준 집계

**한계점**:
- 실제 영업 여부 미반영 (등록 기준)
- 폐업 후 말소 지연 가능성
- 업종 분류 오차 (유사 업종 포함)

---

### 2. 유동인구 데이터 (`grid_traffic`)

**기준**: 2024년 10월

#### 2-1. 지하철 승하차

**출처**:
- 서울교통공사 (1~8호선)
- 서울메트로9호선
- 인천교통공사

**포함 노선**:
- 서울: 1~9호선, 신분당선, 경의중앙선, 분당선 등
- 인천: 1~2호선

**미포함**:
- 경기도 일부 구간 (GTX, 신규 연장 구간)

#### 2-2. 버스 승하차

**출처**: 공공데이터포털 버스정류소별 승하차 인원

**포함 지역**: 서울, 경기, 인천

**한계점**:
- 실제 유동인구가 아닌 대중교통 이용객 기준
- 도보/자가용 이용자 미반영
- 시간대별 데이터는 상권 유형 기반 추정치

---

### 3. 임대료 데이터 (`district_rent`)

**기준**: 2024년 3분기 (2024-Q3)

**출처**: 소상공인시장진흥공단 상가임대료 정보

**원본 파일**: `commercial_rent_2025Q3.xlsx`

**단위**: 만원/평 (3.3㎡ 기준)

**집계 단위**: 구/동 단위 평균

**한계점**:
- 신규 계약 기준 (기존 계약 미반영)
- 대로변/이면도로 구분 없음
- 권리금 미포함

---

### 4. 지하철역 위치 (`subway_stations`)

**기준**: 2024년

**출처**: 국가교통DB, 각 운영사 공개 데이터

**포함 정보**:
- 역명, 노선
- 위경도 좌표

---

### 5. 앵커 시설 (실시간)

**기준**: API 호출 시점 (실시간)

**출처**: 카카오 로컬 API

**포함 시설**:
- 스타벅스 (반경 1km)
- 대형마트 (반경 2km)
- 백화점 (반경 2km)
