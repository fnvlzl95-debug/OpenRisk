## 핵심 요약

### 리스크 점수 구조
```
총점 = Σ(지표 점수 × 업종별 가중치) + 시간대 보정(±10) + 상권 패널티(0~15)
```

### 7대 핵심 지표
1. **경쟁 밀도** - 동종 업종 점포 수
2. **유동인구** - 지하철+버스 승하차 기반
3. **임대료** - 구/동 평균 (만원/평)
4. **생존율** - 폐업률/개업률
5. **앵커 시설** - 역세권, 스타벅스 등
6. **시간대 패턴** - 아침/낮/저녁 분포
7. **상권 유형** - 주거/혼합/상업/특수

### AI 분석 포지션
**"창업을 말리는 AI"**
- 리스크 중심 분석
- 낙관적 해석 금지
- 데이터 기반 경고
