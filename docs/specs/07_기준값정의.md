## 3. 유동인구 레벨

**파일**: `lib/v2/traffic-index.ts:375-381`

> 2026.01 백분위 기반 개편 (평균 6, 중앙값 3, 최대 70)

| 레벨 | 지수 | 백분위 | 의미 |
|------|------|--------|------|
| `very_low` | ≤1 | 하위 25% | 거의 유동인구 없음 |
| `low` | 2~6 | 25~75% | 적음 (일반 주거지) |
| `medium` | 7~13 | 75~90% | 보통 (소규모 상권) |
| `high` | 14~40 | 90~99% | 많음 (역세권/상업지) |
| `very_high` | >40 | 상위 1% | 매우 많음 (강남/명동/가산급) |

### 유동인구 지수 계산 기준

| 파라미터 | 값 | 파일 위치 |
|----------|-----|----------|

| 버스정류장 영향 반경 | 300m | `lib/v2/traffic-index.ts:16` |
| 지하철 승하차 최대값 | 200,000명/일 | `lib/v2/traffic-index.ts:20` |
| 지하철 승하차 최소값 | 5,000명/일 | `lib/v2/traffic-index.ts:21` |
| 버스 승하차 최대값 | 10,000명/일 | `lib/v2/traffic-index.ts:24` |
| 버스 승하차 최소값 | 500명/일 | `lib/v2/traffic-index.ts:25` |

## 11. 리스크 카드 조건

**파일**: `lib/v2/interpretations/risk-cards.ts`

| 카드 | 조건 |
|------|------|
| 경쟁 과열 + 유동 부족 | 동종 ≥ 10개 AND 유동인구 = low |
| 경쟁 과열 + 유동 많음 | 동종 ≥ 10개 AND 유동인구 = high |
| 블루오션 | 동종 = 0개 |
| 과포화 | 동종 ≥ 8개 |
| 고비용 + 경쟁 | 임대료 = high AND 동종 ≥ 8개 |
| 생존 위험 | 폐업위험 = high AND 순증감 < 0 |
| 야간 활성 | 저녁 비율 ≥ 45% |
| 비역세권 | 지하철 > 500m |
| 리스크 집중 | 동종 ≥ 8 AND 임대료 = high AND 생존 = high |

## 수정 가이드

### 기준값 조정 시 고려사항

1. **분석 반경**: 500m는 도보 5분 거리 기준. 업종에 따라 다를 수 있음
2. **경쟁 레벨**: 업종별 시장 특성에 따라 조정 필요
3. **임대료 기준**: 지역별 물가 차이 반영 필요
4. **가중치**: 업종 특성에 따른 세밀한 조정 필요

### 변경 시 영향 파일

- `lib/v2/riskEngine.ts`: 리스크 점수 계산
- `lib/v2/traffic-index.ts`: 유동인구 지수
- `lib/v2/closure-risk.ts`: 폐업 위험 계산
- `lib/v2/interpretations/`: 해석 메시지
- `lib/categories.ts`: 업종별 가중치
- `app/api/v2/analyze/route.ts`: 분석 반경
